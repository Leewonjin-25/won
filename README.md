# ♿ 무장애(Barrier-Free) 문화예술관광 탐방 가이드

> **모든 이들을 위한 장벽 없는 관광 정보 서비스** > 사용자가 지역과 카테고리를 선택하여 배리어프리 관광지를 조회하고, 지도와 목록을 통해 직관적으로 위치를 파악할 수 있는 웹 서비스입니다.

---

## ✨ 핵심 기능 (Key Features)

### 🗺️ 스마트 지도 및 목록 연동
- **위치 이동**: 리스트에서 관광지 이름을 클릭하면 지도가 해당 좌표로 부드럽게 이동(`panTo`)하며 최적의 확대 레벨로 조정됩니다.
- **인터랙티브 UI**: 마우스 호버 효과를 통해 클릭 가능한 항목을 시각적으로 강조합니다.

### 📑 효율적인 데이터 브라우징
- **10개씩 끊어보기**: 초기 검색 시 상위 10개 항목만 노출하여 가독성을 높였습니다.
- **더보기(Load More)**: 페이지 전환 없이 추가 데이터를 하단에 계속해서 불러올 수 있습니다.

### 🗓️ 나만의 여정 관리
- **계획표 만들기**: 관심 있는 장소를 확인한 후, 하단의 '나만의 여행 계획표 만들기' 버튼을 통해 다음 단계로 이동할 수 있습니다.
