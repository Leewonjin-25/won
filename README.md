# 🌉 전국 교량 현황 실시간 검색 시스템

국토교통부에서 제공하는 **공공데이터포털 오픈 API**를 활용하여 전국의 교량 정보를 실시간으로 조회하고 필터링할 수 있는 웹 애플리케이션입니다.



## 🚀 주요 기능
- **실시간 데이터 연동**: 공공데이터포털(Open API)의 최신 교량 현황 데이터를 직접 호출합니다.
- **다중 필터링 시스템**:
  - **지역(시도)**별 검색
  - **노선명(도로명)**별 검색
  - **상부구조형식**별 검색
- **반응형 디자인**: 모바일과 데스크톱 환경 모두에 최적화된 현대적인 UI를 제공합니다.

## 🛠 사용 기술
- **Frontend**: HTML5, CSS3 (Custom Properties), JavaScript (ES6+)
- **API**: 국토교통부 교량 현황 데이터 (JSON)
- **Deployment**: GitHub Pages

## 📂 파일 구조
- `index.html`: 서비스의 메인 구조 및 레이아웃
- `style.css`: 현대적인 카드 스타일 및 테이블 디자인
- `script.js`: API 호출, 데이터 필터링 및 동적 테이블 렌더링 로직
