# 🌉 전국 교량 검색 애플리케이션

국토교통부 오픈 API를 활용하여 전국의 교량 현황을 실시간으로 검색하는 웹 서비스입니다.

## 🚀 기능
- **실시간 API 연동**: 공공데이터포털의 최신 교량 현황 데이터를 불러옵니다.
- **다중 검색**: 지역(시도), 도로명, 상부구조형식을 조합하여 검색 가능합니다.
- **GitHub Pages 배포**: 서버 없이 웹 브라우저만으로 동작합니다.

## 🛠️ 사용 기술
- HTML5, CSS3, JavaScript (Fetch API)
- 공공데이터포털 오픈 API (국토교통부 교량 현황)
